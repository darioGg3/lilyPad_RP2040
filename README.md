# lilyPad_RP2040
 lilyPad style board with an RP2040, it was design in KiCAD 8.0 .

 Characteristics:
 - Powering via USB C or Pins (3.3V - GND).
 - Programing via USB.
 - Debugging port available SWD.
 - 128Mb Flash memory.
 - 16 GPIOs, 3 of them ADC inputs.
 - Programable LED on GPIO13.
